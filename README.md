### Basic Editing 

#### Navigation Pane
cmd+shift+E -> Explorer Pane
cmd+shift+F -> Search Pane 
cmd+shift+G -> Source Control
cmd+shift+D -> Debug Pane 
cmd+shift+X -> Extentions Pane

#### Search all commands
cmd+shift+p -> search all commands

#### Zen Mode
cmd + K + Z -> Zen mode

### Integrated Terminal
control + ` -> Opens Terminal (toggle)
